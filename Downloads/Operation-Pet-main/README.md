## üöÄ Getting Started

### Prerequisites

- **Unity 2020.3 or higher** (for development)
- **Windows OS** (for running the build)
- **Git** (for version control)
- **Git LFS** (for large file storage)

### Installation

#### Option 1: Play the Game (Pre-built)

1. **Clone the repository**
   ```bash
   git clone https://github.com/ShivaShanmukh/Project-PET.git
   cd Project-PET
   ```

2. **Navigate to the build folder**
   ```bash
   cd "Project Pet Build"
   ```

3. **Run the executable**
   ```bash
   "Project Pet.exe"
   ```

#### Option 2: Development Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/ShivaShanmukh/Project-PET.git
   cd Project-PET
   ```

2. **Install Git LFS** (if not already installed)
   ```bash
   git lfs install
   git lfs pull
   ```

3. **Open in Unity**
   - Launch Unity Hub
   - Click "Add" and select the `Operation-Pet` folder
   - Open the project with Unity 2020.3 or higher

4. **Build the project**
   - Go to `File > Build Settings`
   - Select your target platform
   - Click "Build" and choose output directory
